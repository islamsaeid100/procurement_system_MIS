# Purchasing Management System

A full-stack application for managing purchase orders and inventory operations.

## Tech Stack

- **Backend**: Python Django
- **Frontend**: React


## Features

- Order management and tracking
- Inventory monitoring
- Vendor management
- Purchase reporting
- User authentication
